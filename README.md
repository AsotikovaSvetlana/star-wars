## Быстрый старт

- Клонируйте репозиторий: `git clone https://github.com/AsotikovaSvetlana/star-wars.git`
- Перейдите в папку с проектом: `cd star-wars`
- Установите зависимости: `npm install`
- Запустите проект: `npm run dev`
